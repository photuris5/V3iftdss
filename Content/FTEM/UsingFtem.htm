Using FTEM

The FTEM interface is arranged to walk you through the process. Use the tabs to select a wildfire, select your treatment areas, enter monitoring data, and review and submit your data. Below, we walk through each stage of this process. The follow steps assume you have an FTEM account (If not, visit the FTEM Setup Topic for instructions). Upon opening FTEM, you will see wildfires with a split view of on the left, and a table view on the right.

Wildfire:

Pick a wildfire based on year and wildfire name:

  1. Enter a Control Year, the year the wildfire was declared under control, or extinguished.
  2. Use the wildfire search field to further constrain your results by entering the fire name.
  3. Click the fire name from the list
  4. Optional: View the fire on the map, or view past monitoring for that fire
  5. Proceed to the Treatments Tab

Notice that each wildfire listed is followed by information, left to right, describing the number of acres, control date, jurisdiction unit, and the number of interactions.

 Treatments: Select your treatments

If a treatment is within you're agencies jurisdiction you will see an empty check box to the left.

Select only the treatments that interacted with the wildfire.

  1. If you know there are treatments that interacted with the wildfire, but do not see them here, click the Add Treatment(s) button to search and select treatments near the fire (This option will allow you to choose a buffer distance to search within)
  2. If there are treatments that should not be on this list, or did not interact with the fire, use the checkbox to select them and click the Remove Treatment(s) button.
  3. Once you have all desired treatments, select them either individually or by using the Select All button
  4. Proceed to Monitor Tab

Notice that each treatment listed is followed by information, left to right, describing the treatment ID, type of treatment, date of completion, and agency. number of acres, control date, jurisdiction unit, and the number of interactions.

 Monitor: Enter your data

  1. Select the checkbox next to the treatment.
  2. Scroll down and click the monitor button.
  3. A series of headers will appear below the treatment, each one expands to list out specific questions. The first header, Interaction Details and Treatment Effects, is required. Enter data into each field. We briefly describe guidance for data in each field in the FTEM Required Questions Guidance Topic.
  4. Proceed to any optional fields and enter data. We briefly describe guidance for data in each field in the FTEM Optional Questions Guidance Topic.
  5. Once your data is entered, click the save button.
  6. Proceed to the review tab.
 Review: Review and submit your data
