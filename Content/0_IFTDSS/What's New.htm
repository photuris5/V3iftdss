Here is What's New in

Week of 10/3/16 - 10/14/16

IFTDSS team is testing functionality for the new Map Studio, Landscape Evaluation which is similar to Landscape critique in WFDSS and the ability to view Flammap Basic outputs on the Map. IBM will be working on a new set of requirements beginning 10/10 including building Landscape editing tools, creating a fuels treatment comparison dashboard, enhancements to the Map Studio and the beginning of a tool using Landfire lookup tables to help guide a user when changing fuel models.

 Week of 9/26/16

The IFTDSS Team is testing the new cloud environment, engaging with IBM on the design and user interface, and evaluating landscape data including display in tables, charts, and graphs.

 Week of 9/21/16

The IBM Team is working on a Landfire data loading feature, building user workspace and file structure, landscape display in Map Studio, and FlamMap 5.0.

 Week of 9/19/16

IBM is working on loading Landfire data, building user workspace and file structure, display of new landscapes in Map Studio, and building FlamMap 5.0 service.

 Week of 9/6/16

The IFTDSS developers are working on integrating the newest version of FlamMap 5.0, building a new Map Studio, and establishing a linkage that will enable WFDSS reference layers to be accessible to users within IFTDSS

 Week of 8/22/16

The IFTDSS Team and developers from IBM are meeting in Boulder Colorado to examine new modeling capabilities and develop a release schedule for IFTDSS.
