Landscape Tools

Landscape Tools enables you to view or create new landscape files from within . The options available include:

Add to Map

The Add to Map option lists all the available landscape files for display.

 

Once a landscape is chosen by selecting the checkbox next to it, it will become visible, and will automatically zoom to that landscape.

The Filter by Map Extent option allows you to filter your available landscape by their spatial extent, this can be convenient if you have many different landscape files saved in your account.

The Refresh option refreshes the landscape list with the most recent files, this is convenient if you have just downloaded a landscape file but do not see it listed yet.

It may take several minutes after creating a landscape file for it to become visible in the Landscape Tools list of available landscapes.

 

 Create New

The Create New option allows you to download a new landscape file from LANDFIRE.
