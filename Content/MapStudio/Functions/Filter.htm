Filter Button

The Filter button you to limit the visibility of polygons or shapefiles when working in . This can be useful if you are working in an area with many shapefiles (such as many treatments clustered in a small area, or overlapping each other). Only the polygons that meet the expression criteria will be visible in the map.

To add a filter, first go to the Edit Filter tab. At the highest level you can Filter by Folder and then Filter by Name. When Filter by Folder is set to All Folders, all of your shapefiles will be listed. When a specific folder is selected using the pick-list, only those within the folder are listed.

By default all of the shapefiles are selected for filtering. To change the selection press Select to activate the ability to select and unselect the individual shapefiles. If you would like to select all the shapefiles click the Select All button, or to deselect all selected shapefiles click the Deselect button. After you have selected the shapefiles you would like to display click Apply Filter .

To remove your filter and have all shapefile display in Map Studio simply press Remove Filter .
