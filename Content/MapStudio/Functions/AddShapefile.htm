Add a Shapefile

Use the Add Shapefile to add a shapefile in .

  • Select Upload File.
  • Choose your zipped shapefile
  • Select the desired fields from your shapefile using the checkbox next to each field name ( Up to 10)
  • Select Upload File
  • Select Add to Map
Uploaded shapefiles are visible on the map but are not automatically saved to MyWorkspace, they will be lost when you navigate away from unless they are saved to MyWorkspace. See the Topic on Saving Shapefiles to complete this important step.

  • To save the shapefile, make sure to check Save result to my workspace.
  • When finished, click Add to map.
  • To add a shapefile to it must contain shp, .shx, .dbf, and .prj files, and be in a compressed (zipped) file format. Visit the About Shapefiles topic for full details on shapefiles in .
