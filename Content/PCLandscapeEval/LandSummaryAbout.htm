Landscape Summary

The first step in working with landscapes in is to download a Landscape File (.lcp file), and evaluating the information. The Landscape Evaluation stage of the Planning Cycle walks you through the process of creating and summarizing a landscape.

For this topic we will assume you are starting fresh, with no existing landscape files or shapefiles.

From the Landscape Evaluation Stage of the Planning Cycle, select Evaluate/Summarize landscape data layers.

Skip the first two boxes under the 'Select Landscape' section and look at the 'Create a Landscape' Section.

If you know the coordinates for your landscape, and don't need a map interface, you can create your landscape file by selecting the Create Here button.

You may use either the (a) select box button then draw a rectangle over your map area, or (b) enter the coordinates of your area directly into the fields.

c) Select the version of LANDFIRE data to download

d) Select whether to use fire behavior fuel model 13 or 40

e) Specify your spatial resolution (30 meters is the default)

f) Give your landscape a descriptive name

e) Select the folder in MyWorkspace that you want the file saved to

Note: To create a new folder from this interface, select plus icon to the right of the folder field

g) Once the above steps are complete select the Create button, which will turn green when all the needed inputs have been entered.

You will automatically be taken to the landscape summary options page.

A red hourglass icon next to the new landscape means it is still processing. You may need to wait a few seconds and then select the Refresh button

Once the page has refreshed and the landscape file is ready for a summary, a green check mark will appear next to the landscape, and metadata will be visible at the bottom of the page. Select Request a Report.

A summary report will be generated displaying landscape statistics geospatially graphically, and in a tables.

 If you need to use the map to find your landscape or determine its boundaries, create your landscape file by selecting the Create in Map Studio Button.

Selecting Create in will automatically take you to the interface with Landscape Tools open. Use the map to locate your desired area.

Once you select create, a message will appear at the top of Landscape Tools to confirm you landscape has been submitted.

Navigate back to the Landscape Evaluation Stage of the Planning Cycle, select Evaluate/Summarize landscape data layers.

Select your landscape from the My Landscapes dropdown menu

Once the landscape is selected, metadata will be visible at the bottom of the page. Select Request a Report.

A summary report will be generated displaying landscape statistics geospatially graphically, and in a tables.
