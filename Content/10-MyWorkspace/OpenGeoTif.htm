Opening an Geo Tiff file in ArcMap

Downloaded Geo Tiff files from will be .zip files. To begin, move the file to the desired folder on your local computer, and extract the contents.

Next, you may add the Geo Tiff bands to Arcmap. The below steps outline the process in ArcMap version 10.3, your version may vary slightly.

To add the Geo Tiff to ArcMap:

  1. Start by going to the Add Data prompt in ArcMap.

  2. Navigate to your extracted Geo Tiff file.

  3. Double click the raster to reveal individual bands. Each of these bands corresponds to a landscape feature or model output (such as canopy cover, flame length, etc.).

  4. Select all the individual bands and click Open.

  5. The individual bands will now be visible in ArcMap.

For instructions on how to reclass outputs such as flame length in ArcMap, see the Reclassing GeoTiff Features in ArcMap Topic.
