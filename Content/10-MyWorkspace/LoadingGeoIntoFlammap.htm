Loading GeoTiff into FlamMap 5.0

  1. Open FlamMap 5.0 (or newer).
  2. Right-click No Landscape File.

  3. Click Open.

  4. Change the file type to GeoTIFF Landscapes (*.tif).
  5. Select the landscape you want to open.

  6. Click Open.

  7. Once open the landscape has the following units.

    • Elevation (m)
    • Slope (deg)
    • Aspect (deg)
    • Fuel model (categorical)
    • Canopy cover (%)
    • Stand Height (m)
    • Canopy Base Height (m)
    • Canopy Bulk Density (kg/m3)
