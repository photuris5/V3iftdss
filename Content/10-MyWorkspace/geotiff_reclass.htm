Reclassing GeoTiff Features in ArcMap

When working with GeoTiff data in other products, you may notice the data is displayed at as a range of values, not classes. If you need classes, the below instructions outline how to do so by changing symbology, or how to reclassify the data for further analysis. The below example focuses on the flame length, but the same steps are applicable for other outputs as well.

Viewing flame lengths as classes

If you only want to map your flame length output, all you need to do is change the symbology. These changes do not allow you to do further analysis and are not kept if you don’t save your ArcMap.

  1. Add the Flame Length band from the GeoTiff. Using the Add Data button navigate to your GeoTiff, double click the tiff file to show the individual bands. Select "Flame Length".

  2. Right click on the Flame Length layer in ArcMap table of contents, and click Properties.

  3. Go to the Symbology tab in the Layer Properties window, and select Classify. Set the Classes to 7 and click Classify.

  4. In the Classification window you will need to set the break values using the conversion table below to match the bins. To do this you click on the value in the box and enter your break values. Leave the maximum value unchanged (In the image below it is 94.34368896, but your data will have its own max value). Then click "OK", which will close this window, leaving you at the Layer Properties window.

    Flame length (meters) Flame length (feet)
    0 0
    0.3048 1
    1.2192 4
    2.4384 8
    3.3528 11
    7.62 25
    Max value Max value

  5. In the Layer Properties window, change the label to be the feet equivalent. Click "OK" to apply the new symbology.

  6. To change the colors of your new symbols, you can right click on each class and choose the new color.

 Reclassifying flame lengths

To perform further analysis with your flame length data, such as calculating the total area of flame length classes for treatment subunits, you will need to reclassify your flame length output. You will need the Spatial Analyst Extension for ArcGIS and it will need to be turned on.

  1. Verify you have the Spatial Analyst Extension. In the main ArcMap toolbar go to Customize and then select Extensions. When the Extensions window opens, verify Spatial Analyst is listed, available, and is checked.

  2. Add the Flame Length band from the GeoTiff. Using the Add data button, navigate to your GeoTiff, double click the tiff file to show the individual bands. Select "Flame Length".

  3. Open ArcToolbox and navigate to the Reclassify tool by navigating to Spatial Analyst Tools and then Reclass.

  4. When the Reclassify window opens, select the Flame Length band as the Input Raster. Verify the Reclass field is set to "Value" and click Classify.

  5. In the Classification window you will need to set the Classes to 7. Do this step first otherwise you can’t change it once you start setting the Break Values. Then set the Break Values using the conversion table provided below to match the bins. To do this you click on the value in the box and type what you want. Leave the maximum value what it is for your data, not what is in the image. Then click OK.

    Flame length (meters) Flame length (feet)
    0 0
    0.3048 1
    1.2192 4
    2.4384 8
    3.3528 11
    7.62 25
    Max value Max value

  6. In the Reclassify window. Change your New Values to be something meaningful to you to represent the new classification. One suggestion is using the maximum value in the classes they represent.

  7. Go to the Output raster option to name and save it where you would like. Because it is a raster, you are limited to 13 characters for the name and the only special character you can use is an underscore.

  8. When the Reclassify screen opens, click OK to generate your new raster.

  9. You can change the labels by right clicking on the layer name in the table of contents, choosing Properties, and then the Symbology tab. Then typing the Label you would like.

  10. Finally, you can change the colors of your new symbols by right clicking on each class and choosing the color you would like.
