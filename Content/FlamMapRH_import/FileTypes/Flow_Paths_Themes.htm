Flow Paths Themes

Flow path themes are the minimum travel time pathways between the user defined nodes. The density or spacing of the nodes is determined by the Resolution of Calculations setting on either the Minimum Travel Time or Treatment Optimization Model tabs. There are four different flow path themes generated by FlamMap. The two MTT path themes are generated from the Minimum Travel Time tab and the Treat path themes are calculated with the Treatment Optimization Model.  All four themes can be exported as an ESRI shapefile for display in Arc GIS products.

MTT Flow Paths

This theme displays the minimum time pathways between all the nodes on the analysis area.

MTT Major Paths

This theme is a subset of the MTT Flow Paths item showing just the most significant fire spread pathways. The spacing of the MTT Major Paths is determined by the Interval for Minimum Travel Paths (distance): setting on the Minimum Travel Time tab.

Treat Flow Paths

This theme displays the post treatment minimum time pathways between all the nodes on the analysis area. The areas treated are shown by the Treatment grid.

Treat Major Paths

This theme is a subset of the Treat Flow Paths item showing just the most significant post treatment fire spread pathways. The density or spacing of the nodes is determined by the Maximum Treatment Dimension (m): setting on the Treatment Optimization Model tab.
