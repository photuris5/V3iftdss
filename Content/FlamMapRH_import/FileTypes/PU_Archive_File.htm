Archive File (.FZA)

The Archive (.FZA) File saves everything for your FlamMap Project to one compressed file for ease transporting. This feature is very useful for sending a project to someone else, transferring your work between computers, backing up your work, or archiving a completed project. A FlamMap Archive (.FZA) File is created for an open project with the File > Save and Archive command.

The Archive (.FZA) File is compressed in the .zip format and can also be extracted with the WinZip8 utility as well as with the File > Extract and Open Archive command in FlamMap.
