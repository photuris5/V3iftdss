Fuel Conversion File (.CNV)

An optional ASCII text file used in the FARSITE fire area simulator.

A Fuel Conversion (.CNV) File is necessary if the fuel model numbers in the GIS Fuel Model theme do not correspond directly to the fuel numbers in the fuel model file or the standard 13 fuel models and 0, 98 or 99 for non-fuel. It is recommended that you reclassify your Fuel Model theme to make the fuel attributes correspond directly to fuel models (e.g. use the GRASS command r.reclass).

If you don't want to reclassify the fuel attributes, you'll have to use a Fuel Conversion (.CNV) File. For example, fuel types 68, 74, and 86 on the Fuel Model theme may actually correspond to NFFL fuel models (Anderson 1983) 2, 5, and 8, respectively. The Fuel Conversion (.CNV) File would contain the following lines:

68 2
74 5
86 8

Perhaps a better use for a Fuel Conversion (.CNV) File is for changing the fuel model that a particular fuel attribute represents. For example, your fuel map may contain NFFL fuel models 5 and 8 that you decide should really be 6 and 9, respectively (or custom models). You would then create and load a Fuel Conversion (.CNV) File that contains:

5 6
8 9

Each Fuel Conversion (.CNV) File must contain data in the space delimited ASCII format specified below. FlamMap has a Custom Editor to help you create or edit Fuel Conversion (.CNV) Files. Or, a text editor such as Notepad or WordPad, a spreadsheet, or the FlamMap Editor can be used to create or edit ASCII text files.

The format for the Fuel Conversion (.CNV) File is:

ConvertFrom FuelMod

  • ConvertFrom is the index or attribute of a fuel type in the Fuel Model theme converted to a fuel model number 1-99, (integer)

  • FuelMod can be any fuel model 1-50, (integer)

Using the Fuel Model Conversions File Custom Editor

Note: The Fuel Model Conversions (.CNV) File Custom Editor is not currently available in FlamMap. The instructions below are for the FARSITE Custom Editor.

The Fuel Model Conversions (.CNV) File Custom Editor is accessed through the "FARSITE Project" dialog box. The button to the right of the Fuel Model Conversions (.CNV) File text box brings up the custom editor for generating or editing a Fuel Model Conversions (.CNV) File.

Editing an Existing Fuel Model Conversions File

If you currently have a Fuel Model Conversions (.CNV) File loaded, this dialog box will allow you to edit the contents of that file.

The Custom Editor allows direct editing in the text box by selecting text with the cursor, using the backspace key, and typing text similar to a text editor.

You can also use the editing tools in the Custom Editor. First select the range of lines (fuel models) you wish to edit with the cursor. You can use the Add and Set buttons to convert the fuel models selected in the text box.

  • The Add button adds the value displayed in the spin box to all the values in the currently selected lines.

  • The Set button replaces the values in the currently selected lines with the value displayed in the spin box.

Create a New Fuel Model Conversions File

To generate a new Fuel Model Conversions (.CNV) File click the New .CNV File button. A new file with fuel models 1-50 is displayed in the text box. You can edit this data as shown above.

Finally

Before clicking the Apply & Exit button make sure the modified file has been saved with the Save .CNV File button. If you don't save the changes, they will still be in effect for the current simulation, but they will be lost when the simulation is terminated. However if the Project (.FPJ) File is saved with an unsaved edited Fuel Model Conversions (.CNV) File, the Fuel Model Conversions (.CNV) File will then be automatically saved also.
