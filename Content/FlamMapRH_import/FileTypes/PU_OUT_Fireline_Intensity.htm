Fireline Intensity

One of the output grid themes. Units can be either kW/m or BTU/ft/sec. This output grid can be calculated in many different ways depending on the options and settings selected.

Fireline Intensity Grid

When selected on the Fire Behavior Outputs tab the grid displays the fireline intensity for the spread direction specified in the Options section of the Fire Behavior Outputs tab. For the default Options section settings (Relative Spread Direction From Maximum and zero degrees) the fireline intensity in the direction of the maximum rate of spread is calculated.

MTT FLI Grid

When selected on the Minimum Travel Time tab this grid displays the fireline intensity as it encountered each node in the direction of fire spread along the minimum travel time path.
