Dead Fuel Moisture Content Grids

The 1, 10, 100, & 1000 hour Timelag Dead Fuel Moisture Content output grids can be created only if using a fuel moisture conditioning period as specified on the Inputs tab of the "Run:" dialog box. Each grid contains the dead fuel moisture for each cell at the end of the optional fuel moisture conditioning period in fractional format (e.g. 0.11).

The resolution and extent of the raster output file is the same as that of the source Landscape (.LCP) File for the run.

The dead fuel moisture model is described in the Dead Fuel Moisture Technical Reference.
