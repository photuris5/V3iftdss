Fuel Model Theme

The raster fuel model theme must specify an integer index to a fuel model. Model numbers 1-13 are reserved for the original standard NFFL fuel models (Anderson, 1983) and some numbers between 101-219 are used for the expanded set of standard models (Scott and Burgan, 2005). Water is represented by number 98 and rock as 99. Other non-burnable values include 90-97. See Table 1, Scott and Burgan, (2005) for a complete list of standard fuel model number assignments.

Within the United States, pre-made Fuel Model Themes can be found at www.landfire.gov.

The presence of custom fuel models and/or fuel model conversions in the fuel model theme should be indicated when generating a Landscape (.LCP) File. This is done in the "Landscape (.LCP) File Generation" dialog box, in the Source Units text box for the Fuel Model: theme there are four options available from the drop down list;

  • Class
  • Custom
  • Convert
  • Custom +Convert

Class

Indicates the fuel model theme contains only fuel model numbers relating directly to the standard fuel models (original 13, Scott and Burgan set, or 0, 98 or 99 for non-fuel).

Custom

Indicates custom fuel models are used (see Custom Fuel Model (.FMD) File) in the fuel model theme. The Custom Fuel Model (.FMD) File is identified on the Inputs tab of the "Run:" dialog box.

Any fuel model other than the standard fuel models are custom fuel models. These must be assigned model numbers between 1 and 256 NOT already assigned to a standard fuel model. Although FARSITE and FlamMap will permit any custom fuel model number between 1 and 256 not assigned to a standard fuel model it is best to limit custom model numbers to the range 14-89 to limit conflicts with future standard models.

Convert

NOTE: FlamMap DOES NOT support fuel model conversions. It is included in the FlamMap "Landscape (LCP) File Generation" dialog box only to make the .LCP files compatible with FARSITE.

Indicates conversion between fuel indices and fuel model numbers is necessary (see Fuel Conversion (.CNV) File). A fuel index is the integer number describing fuel types in the raster Fuel Model theme. This index must be converted to a fuel model number if:

  1. it exceeds 256,

  2. it is one of the standard numbers which does not relate to one of the standard Fire Behavior fuel models.

For example, a Fuel Model theme with all indices less than 14 would require selecting only Convert if all of these did not correspond directly to the 13 NFFL fuel model numbers or 0, 98 or 99 for non-fuel. Select Custom + Convert if any indices greater than 50 correspond to custom models. If all indices were between 14 and 50 and corresponded to directly to the custom model number, select only Custom from the drop down list.

Custom + Convert:

Selecting Custom + Convert from the drop down menu indicates that custom fuel model indices and are in the fuel model theme and fuel model conversions will be required. The Custom Fuel Model (.FMD) File and Fuel Conversion (.CNV) Files are identified on the Inputs tab of the "Run:" dialog box.
