rate of spread

One of the output grid themes. Units can be m/min, ft/min, or Chains/hr. This output grid can be calculated in many different ways depending on the options and settings selected.

Rate of Spread Grid

When selected on the Fire Behavior Outputs tab the grid displays the rate of spread in the direction specified in the Options section of the Fire Behavior Outputs tab. For the default Options section settings (Relative Spread Direction From Maximum and zero degrees) the maximum rate of spread for each grid cell is calculated.

MTT ROS Grid

When selected on the Minimum Travel Time tab this rate of spread grid displays the spread rate of the fire as it encountered each node in the direction along the minimum travel time path.
