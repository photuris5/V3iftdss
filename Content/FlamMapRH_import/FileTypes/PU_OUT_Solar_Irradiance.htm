Solar Radiation

The solar radiation output grid theme can be created only if using a fuel moisture conditioning period is specified on the Inputs tab of the "Run:" dialog box. It contains the solar radiation of each cell for the ending time of the optional fuel moisture conditioning period. It is expressed as Watts per square meter (W/m2).

The resolution and extent of the raster output file is the same as that of the source Landscape (.LCP) File for the run.
