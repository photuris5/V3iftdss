10hr Fuel Moisture

The 10 hr fuel moisture output grid theme can be created only if using a fuel moisture conditioning period is specified on the Inputs tab of the "Run:" dialog box. It contains the 10 hr time lag dead fuel moisture for each cell at the end of the optional fuel moisture conditioning period in fractional format (e.g. 0.18).

The resolution and extent of the raster output file is the same as that of the source Landscape (.LCP) File for the run.

The dead fuel moisture model is described in the Dead Fuel Moisture Technical Reference.
