Project File (.FMP)

A FlamMap Project (.FMP) File contains most of the inputs and customizations that you've made for a analysis. A project file is used to speed up the process of loading files and setting the customizations for subsequent analyses.

A Project (.FMP) File contains pointers to all of the data themes and files for FlamMap, as well as run settings and display customizations set at the time the project file was made.

You can also create an Archive (.FZA) File that saves everything from your FlamMap Project to one compressed file for easy transporting or archiving.
