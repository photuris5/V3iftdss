Custom Fuel Model File (.FMD)

This is a optional ASCII text file for utilizing custom fuel models in FARSITE and FlamMap. This is the same format used in the FARSITE v4.1 fire area simulator, so Custom Fuel Model (.FMD) Files are interchangeable between the two applications.

NOTE: The Custom Fuel Model (.FMD) File format is different for FARSITE versions 4.0 and 4.1. For version 4.1 the format now permits a fuel model number up to 256 and adds fields for the fuel model code, fuel model type (static or dynamic), and fuel model name.

  • FlamMap v5.0 only accepts FARSITE version 4.1 format Custom Fuel Model (.FMD) Files.

  • FARSITE v4.1 will convert v4.0 Custom Fuel Model (.FMD) Files to the new v4.1 format.

Fuel models other than the standard 13 Fire Behavior models (Anderson 1982) or the expanded set (Scott and Burgan, 2005) must be supplied in a Custom Fuel Model (.FMD) File.

FARSITE and FlamMap allow custom fuel model inputs in English or metric units. The units are selected by inserting the word ENGLISH or METRIC as the first line of the Custom Fuel Model (.FMD) File. If no option is found English units are assumed.

Each Custom Fuel Model (.FMD) File must contain data in the space delimited ASCII format specified below. FARSITE has a Custom Fuel Model Editor to help you create or edit Custom Fuel Model (.FMD) Files. Also, a generic text editing application such as Notepad or WordPad, a spreadsheet, the FARSITE or FlamMap Editors can be used to create or edit the Custom Fuel Model (.FMD) File. The Custom Fuel Model (.FMD) File must be in the following space delimited format, and have fuel model numbers between 1 and 256 not assigned to a standard fuel model. Although FARSITE and FlamMap will permit any custom fuel model number between 1 and 256 not assigned to a standard fuel model it is best to limit custom model numbers to the range 14-89 to limit conflicts with future standard models. See Table 1, Scott and Burgan, (2005) for a complete list of standard fuel model number assignments.

Using the FlamMap Text Editor

The pencil button, found to the right of the Use Custom Fuels (*.fmd) text box on the Inputs tab of the "Run:" dialog box, opens the Custom Fuel Model (.FMD) File named in the associated text box.
