Canopy Cover Theme

A required raster file for generating a FlamMap/FARSITE Landscape (.LCP) File.

The canopy cover theme is necessary for computing wind reduction factors and shading in the optional dead fuel moisture model. Canopy cover is the horizontal percentage of the ground surface that is covered by tree crowns. It is a stand-level descriptor.

Canopy Cover is NOT the same as crown closure. Canopy cover is measured as the horizontal fraction of the ground that is covered directly overhead by tree canopy. Crown closure refers to the ecological condition of relative tree crown density. Stands can be said to be "closed" to recruitment of canopy trees but still only have 40% or 50% canopy cover.

Coverage units can be categories (1-4) or percentage values (0-100). Categories are assumed to be the following:

  1. 1-20%
  2. 21-50%
  3. 50-80%
  4. 81-100%

with zero cover specified by 0 or 99.

Canopy Cover can also be described as a constant across the entire landscape when generating a Landscape (.LCP) File with the "Landscape (LCP) File Generation" dialog box.

Within the United States, pre-made Canopy Base Height Themes can be found at www.landfire.gov
