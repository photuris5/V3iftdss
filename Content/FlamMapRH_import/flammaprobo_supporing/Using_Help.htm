FlamMap is Different

Using FlamMap is different from most Windows applications. Instead of utilizing menus, commands, and toolbar buttons, FlamMap uses an expanding tree structure in the left hand pane of the "Project" window to guide you through your work.

Menu commands and toolbar buttons are available for most operations but this help and tutorial system will focus on using context sensitive short cut menus activated when you right-click in the project window.

Help System Organization

The FlamMap help system is composed of five main parts, shown as "books" in the Help System navigation pane. They are;

  1. Using the left hand ("Tree") pane for analysis,

  2. Displaying results in the right hand ("Display") pane,

  3. Tutorials, with Basic and Advanced lessons,

  4. A Reference Section,

  5. Technical Topics.

The "Tree" Pane (Left Hand Pane)

The left hand pane of the "Project" window contains the FlamMap "tree" which organizes all the files and parts of your FlamMap Project. Similar to Windows Explorer, the FlamMap tree can be expanded by clicking a where displayed. A indicates that section can be collapsed by clicking. Right-clicking any of the tree items brings up a shortcut menu showing the commands available for that tree item.

Whenever you see this "Tree" pane, as shown below, in the Help System you can use it to find the assistance you need. The white tree items are those found in the help section that you have currently displayed, while the greyed tree items are found in other sections. Regardless of whether an item is white or grey you can roll your mouse cursor over the item and click on it as it is selected. The Help System will then display the section concerning the tree item you selected.















Likewise, many of the shortcut menu images are linked to Help System information.





When you roll your mouse cursor over a shortcut menu image in the Help System if the command changes color (selected), then clicking on that selected command displays the help information for that command.

The "Display" Pane (Right Hand Pane)

The right hand or "Display" pane of the "Project" window displays the active theme selected in the "Tree" pane. This display can be zoomed, colors changed, legends displayed, etc.

The Tutorial

The Basic Lessons are a quick way to learn FlamMap. It takes you through a simple FlamMap run as well as displaying, saving, and exporting. All files and data needed are included when you install FlamMap.

The Advanced Lessons introduce the user to,

  • Variable dead fuel moistures using a conditioning period.

  • Minimum Travel Time (MTT) fire growth methods.

  • The Treatment Optimization Model (TOM) for identifying fuel treatment areas.

The Reference Section

The reference section is composed of file formats and descriptions of data needed to run FlamMap. It is designed to be referenced from the other three sections to provide detailed information on acquiring data to utilize FlamMap.

Technical Topics

In this section you will find information explaining the background of some of the tools and models in FlamMap.

Format of the FlamMap Online Help

Numerous items and instructions appear throughout this guide. These following conventions are designed to make it easy to find and understand FlamMap Online Help.

  • Menu names, options, commands, tree items, and buttons are printed in bold type. i.e. click Open.
