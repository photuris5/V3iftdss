FlamMap Text Editor

Some of the data for FlamMap analyses (known as "Runs") is entered via ASCII text files. These files contain non-spatial weather, fuel model, and fuel moisture data. These files can be created and edited in any available text editor. FlamMap also has a simple text editor, similar to Notepad, where you can view, edit, and save these files from within the application.

The pencil button, found in several places on the Inputs tab of the "Run:" dialog box, opens the file named in the associated text box.

If no file is loaded a new blank file is opened.

Text Editor Buttons

A blank FlamMap Text Editor window can be opened with the New File button.

You can also load a new file into the editor with the Open File button, the new file will be loaded in the Inputs tab of the "Run:" dialog box when you exit the editor.

The Save As button opens a "Save As" dialog where you can save the current text as a new file.

The Inspector button will check the current edits for validity.

The Help button will bring you to the topic associated with the text file above in this online help system.

 If any changes are made with the editor you will be asked if you want to save these changes when you exit the editor by clicking the Close button.
