Open a Landscape

Landscape (.LCP) Files are created with the File > Create Landscape (.LCP) File... command. You can also use your Landscape (.LCP) Files from version 4 of FARSITE. You can download FARSITE v4.04 from www.farsite.org.

If you have an existing FARSITE Landscape (.LCP) File make sure it is the version 4 format. Version 3 Landscape (.LCP) Files can be updated in FARSITE v4.xx with the Input > Landscape Utilities > Convert Landscape File version 3 to 4 command.















Before you Open a Landscape

Before you open a Landscape (.LCP) File you need a empty FlamMap Project open. You can use the default FlamMap1 Project that automatically opens when you start FlamMap or select the File > New command to open a new empty FlamMap Project. An empty FlamMap Project window will look like this;

New projects have a default name of FlamMap2, FlamMap3, ... etc. until you rename them by saving them with the File > Save As command.

Open a Landscape

Open a Landscape (.LCP) File with the File > Open command or right-clicking on the ?No Landscape File item and selecting Open from the shortcut menu as shown below:

A standard Window's "Open" dialog box is displayed. Navigate to and select the desired Landscape (.LCP) or GeoTIFF (.TIF) File and click the Open button. Your landscape will now be displayed in the "Display" Pane with the Fuel Model theme as the default active theme.

With FlamMap one of the Grid themes is always the active theme and is shown in the "Display" Pane. The active theme can be one of the themes in the Landscape (.LCP) File or one of the Auxiliary Grid themes. The active theme is identified in the "Tree" Pane with the  icon.
