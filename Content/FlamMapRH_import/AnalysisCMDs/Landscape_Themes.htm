Landscape Themes

The landscape consists of between five required and five optional themes, each with their own line in the "Tree" Pane. These themes are;

  • Elevation (required)
  • Slope (required)
  • Aspect (required)
  • Fuel Model (required)
  • Canopy Cover (required)
  • Stand Height (optional for torching, spotting, and crown fire models)
  • Crown Base Height (optional for torching, spotting, and crown fire models)
  • Crown Bulk Density (optional for torching, spotting, and crown fire models)
  • Coarse Woody Fuels (optional, not yet utilized in FlamMap)
  • Duff Loading (optional, not yet utilized in FlamMap)

If any of the five optional themes are not available, they can be set globally (the same value across the entire ) with a constant in the Landscape (.LCP) File Generation dialog box. Constants for stand height, crown base height, and canopy bulk density can also be entered at run time on the Inputs tab of the "Run:" dialog box if no values are available from the Landscape (.LCP) File.

Right-clicking any individual tree item (i.e. Canopy Cover) under the Themes item in the "Tree" Pane displays this short cut menu.

Creating Contours

Contours can be created from any landscape, auxiliary, or output grid file by right-clicking a grid theme in the "Tree Pane" or right-clicking the "Display" pane, then selecting Contour from the shortcut menu to display the "Contouring" dialog box.

The Theme Value Range gives the range of values over the theme for the entire landscape. You cannot edit this range.

You can set the contour interval in the Contour Interval spin box. The units are whatever the grid theme is currently displayed in.

The Minimum Distance between points on a contour line: spin box controls how smooth you contour is. Smaller values create smoother contours.

Click the OK button and the contours will be generated and displayed in the default black color.

To temporally hide or display an individual contour theme, simply double click it's line in the left pane tree. A blue check mark "" indicates the theme is currently displayed and a red "X" indicates it is currently hidden.

Save a Landscape File Grid

You can save your grids in several formats so you can use it in a GIS, image editor, or word processor. Right-click an individual grid theme from the "Tree" pane and select Save As… A standard Windows "Save As" dialog box appears where you can navigate and name your new file.

In addition to saving your grid in ASCII Grid format, Bitmap, JPEG, and TIFF image formats are also available from the Save as Type: drop-down list for using your grid as an image in an image or word processor.

Theme Properties

Selecting the Properties command displays the "Create/Modify Legend" dialog box where you can change the colors and number of categories displayed.

Statistics

The Statistics command displays the "Theme Statistics" dialog which consists of two tabs. The first is the General Statistics tab which displays general information about the grid theme and summary statistics.

NOTE: For themes with class units (like the fuel model example shown) some of the summary statistics are meaningless.

The Save As button opens a "Save As" dialog box for the user to save the tab contents in a plain text (.TXT) format file.

The second Classes tab displays the legend bins as determined by the "Create/Modify Legend" dialog box. Cell frequency and area data is also shown.

The Save As button opens a "Save As" dialog box for the user to save the tab contents in a comma separated value (.CSV) format file which can be imported into a text editor or spreadsheet application.
