Generate Landscape File

The Landscape File (LCP) is generated by importing raster files using this "Landscape (LCP) File Generation" dialog box which is opened with the File > Generate Landscape (.LCP) File command.

General Section

To create a new Landscape (.LCP) File from scratch first enter the latitude (whole degrees) of the themes in the Latitude spin box. Then select the units (Meters, Feet, or Kilometers) of the raster cells from the Grid Distance Units drop-down list.

Required Themes

To model surface fire in FlamMap five spatial themes (elevation, slope, aspect, fuel model, and canopy cover) are required in the Landscape (.LCP) File. An entry, either loading a grid or a constant, is needed for each of the required themes. To load a grid theme click the button to the right of each theme Source text box brings up a "Select Grid File" dialog box.

Navigate to the correct grid file, make sure the file name is displayed in the File Name: text box and click Open.

The file name selected will be displayed in the Source text box of the "Landscape (LCP) File Generation" dialog box. Select the correct units for each theme from the drop-down list to the right of the theme file name. Be sure these correctly describe the units of your data in each grid theme before generating the Landscape (.LCP) File; strange displays and weird run output can result from incorrectly setting the units in the Landscape (.LCP) File.

If your units are something other than listed for that theme, you will have to reclassify or recompute values for that theme in a GIS before loading the theme. The Fuel Model theme also requires you to select if custom fuel models are present and/or if fuel model conversions will be needed.

A grid file or constant for elevation, slope, aspect, fuel model, and canopy cover, must be selected before the specified Landscape (.LCP) File can be created. A Landscape (.LCP) File cannot be generated without having a entry for all five required themes.

Crown Fuel Themes

To utilize the torching, spotting, and crown fire model in FlamMap three additional Crown Fuel themes or constants are needed. To load data in the Crown Fuels section, first select the Include Crown Fuels check box. Then add the Stand Height, Canopy Base Height, and Canopy Bulk Density themes or constants as in the Required Themes group.

Ground Fuels Themes

The post-frontal combustion model is not yet available in FlamMap. Adding the additional fuels themes in a FlamMap generated Landscape (.LCP) File is provided so as to make the Landscape (.LCP) File useable in FARSITE where the post-frontal combustion model is available.

To include the duff and large dead fuels themes in a Landscape (.LCP) File, first select the Include Ground Fuels check box. Then add the Duff Loading and Woody Ground Fuel themes or constants as in the Required Themes group.

Constant Values

You have the option of using constant values for any of the GIS themes. By selecting the Constant check box at the far right of each theme, you can use spin box to insert constant values across the landscape instead of providing a grid theme. This can be useful for:

  1. describing very uniform fuel or vegetation conditions,

  2. describing the dominant fuel or vegetation coverage and then using the Landscape Editor in FARSITE v.4 to change small patches of fuel or vegetation conditions,

  3. providing for later use of the FARSITE Landscape Editor when fuel or vegetation conditions are better known. For example, Crown Base Height (CBH) is not typically available. It can be observed in the field for critical areas and then edited in the Landscape (.LCP) File only if information on CBH has been provided to the Landscape (.LCP) File using a raster file or constant value.

Saving the New LCP File

Once you are satisified with the themes and settings selected you need to save as a Landscape (.LCP) File by clicking the Save As LCP button at the top of the dialog box. Remember that at a minimum the General and Required Themes groups need entries.

Revising an Existing LCP File

  1. Click the Load Source LCP File button (upper left in the "Landscape (LCP) File Generation" dialog box). A "Select Landscape File to Load" dialog box appears.

  2. Select the Landscape (.LCP) File to modify and click the Open button. To change the units of an existing theme simply select the correct one from the drop-down list.

  3. To substitute a new grid file for the existing data click the button next to the existing data file and the "Select Grid File" dialog box appears. Select the new file and click the OPEN button.

  4. When finished adding the revised files click the Save as LCP button to display the "Select Destination Landscape File Name" dialog box. Enter a different output Landscape (.LCP) File name if you want to retain the original Landscape (.LCP) File or select the original file name if you want to overwrite the file.

  5. Press the button to close the "Landscape (LCP) File Generation" dialog box when finished.

Editing an Individual Landscape Theme

Tools to edit the individual themes of a Landscape (.LCP) File are not available in FlamMap. However the Landscape (.LCP) Files created in FlamMap are the same as those created in FARSITE v.4. You can modify landscape themes in FARSITE v.4 with the Inputs > Landscape Utilities > Landscape Calculator and Inputs > Landscape Utilities >  Edit Landscape commands.
