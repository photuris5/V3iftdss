Basic Lesson 1

Load Landscape and Themes

The data files for this tutorial are found in your ...FlamMap3\Tutorial\ folder (directory). The ...FlamMap3\  folder is where you chose to place the FlamMap programs during installation. The default folder for installing FlamMap on Forest Service computers is C:\fsapps\fsprod\fam\FlamMap3\. The default for most other computers would be C:\Program Files\FlamMap3\.

When you first start FlamMap it opens a default project window named "FlamMap1".

The right ("Display") pane is empty since no Landscape (.LCP) File is loaded and the left ("Tree") pane contains only two lines, the project and landscape items.

Load a Landscape (.LCP) File

The first step is to load a Landscape (.LCP) File into the default project; right-clicking on the ?No Landscape File item brings up a shortcut menu with just the Open and Help commands.

Select the Open command from the shortcut menu to open a standard Window's "Open" dialog box. Navigate to the ...\FlamMap3\Tutorial\ folder and select Tutorial.LCP as your Landscape (.LCP) File and click the Open button. Your "FlamMap1" project window will look like this.

NOTE: Opening window and pane sizes will vary on each computer. You can change window sizes by dragging the Window Borders. You may also want to drag the divider between the "Tree" and "Display" panes right to see the entire "Tree" pane.   

Load Auxiliary Themes

Auxiliary themes are not required to generate FlamMap output, but they are handy for orienting and viewing results.

Vector Themes

For this lesson you will load two vector shapefiles, one of the major rivers, and another of a recent fire. First, expand the Auxiliary Themes item by clicking it's box. Then right click the Vector Themes tree item to bring up the following short cut menu.

Select the New Vector Theme command from the shortcut menu to display a standard Window's "Open" dialog box. Navigate to the ...\FlamMap3\Tutorial\ folder and select the rivers-major.shp file and click the Open button. Your "FlamMap1" project window displays the rivers on your fuel model layer in the "Display" pane. Now expand the Vector Themes tree item by clicking it's box to show the rivers-major item in the tree.

Again right click the Vector Themes tree item and select the New Vector Theme command from the shortcut menu to display the "Open" dialog box. Navigate to the ...\FlamMap3\Tutorial\ folder and select the whole712.shp file (a recent fire) and click the Open button. Your "FlamMap1" project window displays the rivers and a polygon of the recent fire on your fuel model layer in the display pane.

Grid Themes

Now load a grid (or raster) theme of the vegetative cover type. Right click the Grid Themes tree item to bring up the following short cut menu.

Select the New Grid Theme command from the shortcut menu to display the "Open" dialog box. Navigate to the ...\FlamMap3\Tutorial\ folder and select the cover_type.asc file and click the Open button. Expand the Grid Themes tree item by clicking it's box and select the cover_type item under to display this new grid theme.

Saving a FlamMap Project

At this point in the tutorial or at any place in the tutorial you can save your FlamMap project so you can close FlamMap and return without losing any of your work. To save, right-click on the FlamMap1 line in the tree pane (it's the top line) and select Save from the shortcut menu. A "Save As" dialog box opens with the ...\FlamMap3\Tutorial\ folder selected. Name your project "Tutorial.fmp" and click the OK button.

The next lesson will teach you to view and change the legends of the grid and vector themes.
