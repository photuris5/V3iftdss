Variable Fuel Moistures

The basic "First Tutorial" run uses the fuel moistures from the broad.fms file. The fuel moistures in the Initial Fuel Moistures (.FMS) File can vary only with the fuel model. In reality dead fuel moistures vary widely with topography and shading from clouds and tree canopy. So FlamMap has a  Fuel Moisture Conditioning Period feature that varies the dead fuel moistures for the different environmental conditions within the landscape. The Fuel Moisture Conditioning Period can be used when calculating any of the FlamMap products, Fire Behavior Output, MTT fire growth, Burn Probabilities, or Treatment Optimization.

Weather (.WTR) and Wind (.WND) Files included with the FlamMap installation in the Tutorial folder so in this lesson you can see how FlamMap's dead fuel moisture model affects fire behavior.

Open your Project

If you have closed your project or shut down FlamMap, you can reopen the project by selecting the File > Open command from the menu bar. In the "Open" dialog box select the file Tutorial.fmp and click the Open button. Your FlamMap project will display just the way it was when you last saved it.

Create a New Run

Create a new run, by right-clicking the Runs tree item and select New Run from the shortcut menu. We will keep all the inputs the same except use a Fuel Moisture Conditioning period. Then you can compare the outputs between the two runs.

Inputs Tab

  • Name the new run "Variable Fuel Moistures".

  • Load the broad.fms file for the Fuel Moisture File. You still need a Fuel Moisture (.fms) File to provide a starting point for the dead fuel moisture model.

  • Select the Wind Blowing Uphill radio button and enter a wind speed of 35 mph.

  • In the Canopy Characteristics group make sure the foliar moisture content is 100%. The other three spin boxes should be dimmed.

  • In the Fuel Moisture Settings area select the Use Fuel Moisture Conditioning radio button, open broad.wtr for the Weather (.WTR) File and broad.wnd for the Wind (.WND) File by clicking the button.

  • In the Fuel Moisture Conditioning Period area select a start day/time of 7/10 12:00 PM and an end time of 7/13 16:00 PM. The dead fuel moistures used for the run will be those calculated at the specified End day & time, they DO NOT vary with time as in FARSITE. Live fuel moistures are not affected by the conditioning period, so the live fuel moistures found in the Fuel Moisture (.fms) File will be used in the fire behavior calculations.

The Inputs tab should now look like this.

Outputs Tab

Select the following from the Outputs Grids area,

  • Rate of Spread checkbox,

  • Flame Length checkbox,

  • Crown Fire Activity checkbox.

From the Non-Fixed Fuel Moisture Outputs area select

  • 1hr Fuel Moisture checkbox.

Stick with the defaults for the Processor Usage, Memory/Disk Usage, and Options groups. Click the Apply button and the status bar should indicate Inputs OK and 4 outputs selected.

Launch the Variable Fuel Moistures Run

Click the Launch button and the "FlamMap Progress" status box appears for several seconds. When the fire behavior calculations are complete click OK in the information box and click OK in the "Run:Variable Fuel Moistures" dialog box.

View Outputs

There are now two separate FlamMap runs for the same landscape, the "First Tutorial" run using fixed dead fuel moistures and the "Variable Fuel Moistures" run which calculate fuel moisture with the dead fuel moisture model.

Hide the contours by double clicking the Elevation contour in the "Tree" pane. Now expand the Variable Fuel Moistures line in the "Tree" pane by clicking it's icon and your "Tree" pane will now look similar to this.

 

Alternately select the Rate of Spread grid under the "First Tutorial" and "Variable Fuel Moistures" runs to view differences as shown below. The darker the color the higher the rate of spread.

Rate of spread grid using fixed dead fuel moistures. Rate of spread grid using variable fuel moistures from the dead fuel moisture model.

This dramatic difference can also be seen in the Flame Length and Crown Fire Activity grids. Now select the 1hr Fuel Moisture item under the "Variable Fuel Moistures" run as the active theme.

Notice the variation of 1 hr dead fuel moistures and compare this grid with the constant value of 0.04 used for the fixed fuel moisture ("First Tutorial") run.

You're Finished

You now have completed this lesson showing how to use the dead fuel moisture model. You should save your project so you can refer to it later.
