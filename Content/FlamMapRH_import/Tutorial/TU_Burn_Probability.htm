Burn Probability

The minimum travel time (MTT) feature is a two-dimensional fire growth model.  It calculates fire growth and behavior by searching for the set of pathways with minimum fire spread times from point, line or polygon ignition sources. MTT has a wide variety of uses, in this lesson you will learn how to use MTT to calculate a conditional burn probability for a landscape under a constant set of wind and fuel moisture conditions. Burn probabilities can be useful for comparing the effectiveness of fuel or hazard reduction projects.

A separate lesson using the MTT two-dimensional fire growth feature is also available, it is not necessary to complete this lesson before doing the MTT fire growth lesson.

The Minimum Travel Time (MTT) and Treatment Optimization features in FlamMap v. 3 can be very computer intensive. However this tutorial is designed to work in a reasonable time on a computer with the recommended requirements found on the FlamMap website. (Pentium 4 processor, 256 MB RAM, and Windows 2000 operating system)

The data files for this tutorial are found in your ...FlamMap3\Tutorial\Schley\ folder (directory).

When you first start FlamMap it opens a default project window named "FlamMap1". If you have been working on other FlamMap projects, create an new empty project window with the File > New command. It will have a name of "FlamMap2", "FlamMap3", etc.

Load a Landscape (.LCP) File

The first step is to load a Landscape (.LCP) File into the empty project; right-clicking on the ?No Landscape File item brings up a shortcut menu with just the Open and Help commands.

Select the Open command from the shortcut menu to open a standard Window's "Open" dialog box. Navigate to the ...FlamMap3\Tutorial\Schley\ folder and select schley_pre.LCP as your Landscape (.LCP) File and click the Open button. Your project window will look like this. To match the fuel model colors used in this tutorial select the Standard13 predefined legend.

The Inputs Tab

For an MTT run using the Inputs Tab is identical to other FlamMap runs. Like all FlamMap runs you can utilize wind grids and conditioning periods, but this lesson will use simple wind and weather inputs. Fill in the Inputs tab as shown below.

It is important to emphasize one of the major FlamMap assumptions at this point; all wind and fuel moisture conditions are constant. When a conditioning period is not utilized they do not even vary by aspect, elevation, or shading. In this run FlamMap will use the fuel moisture values in the Fuel Moisture (.fms) File for all calculations.

Make sure to click the Apply button before moving on to the next step.

The Minimum Travel Time Tab

Next select the Minimum Travel Time tab in the "Run:" dialog box.

Ignitions

Select the Random radio button in the Ignitions section.

Inputs

  • Set the Resolution of calculations to 60.

  • Set the Maximum Simulation Time to 600 minutes to simulate a two day fire with a five hour per day burning period.

Outputs

The Burn Probabilities grid is the only available output when the Random radio button is selected.

Launch

Now click the Apply button and the Launch button become active if the required inputs and settings have been made. However making sure the inputs and settings are correct is the responsibility of the user.

Click the Launch button to begin the run and display the "FlamMap Progress..." dialog box.

Outputs

Close the "FlamMap Progress..." dialog box, the "Run:" dialog box, and expand the Basic MTT run item (by clicking the icon) in the "Tree" pane to show the output items. Select the Burn Probabilities item so it is displayed.

The Max Spread Direction and Elliptical Dimensions outputs are designed as inputs for other applications and will not be discussed in this lesson. You can find out more on these grids by clicking on the above links.

The default random colors don't show much information for the Burn Probabilities item so the Legend needs to be changed. Right click on the Burn Probabilities item and select the Properties command from the shortcut menu to display the "Create/Modify Legend" dialog box.

From the Color Ramp: drop-down list, select one of the available ramps, a three color purple/grey/pink ramp is used for the example. Click the Apply button next to the selected ramp and the "Create/Modify Legend" dialog box will look similar the example below.

For more on MTT Burn Probabilities feature see the Random Ignitions section in the Minimum Travel Time Tab topic and the Technical Topic, Minimum Travel Time.

Extra Credit

If you want to compare this burn probability run with one from a "treated" landscape you can use the schley_max.LCP, also found in the ...FlamMap3\Tutorial\Schley\ folder, as your Landscape (.LCP) File. Make sure you keep all the settings the same and use the same legend to view the outputs.
