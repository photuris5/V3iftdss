Basic Lesson 4

Input a Run

This lesson will lead you through a simple FlamMap run where you will create flame length, rate-of-spread (ROS), and crown fire activity maps. While this lesson shows you the simplest way to generate FlamMap output, the optional input and output settings are also easy. In the optional last lesson of the tutorial wind (.WND) and weather (.WTR) files are provided so you can utilize the fuel moisture model in FlamMap and produce additional fuel moisture output maps.

Open your Project

If you have closed your project or shut down FlamMap, you can reopen the project by selecting the File > Open command from the menu bar. In the "Open" dialog box select the file Tutorial.fmp and click the Open button. Your FlamMap project will display just the way it was when you last saved it.

Do some changes to your project window as shown below. Make the Fuel Model theme the active grid theme and make the "tree" pane taller so there is some space below the last item. You will be adding more items to the tree in this lesson.

The Inputs Tab

Right-click on the Runs line (currently the bottom item) in the "Tree" pane to display this shortcut menu:

Select the New Run command from the shortcut menu to open the "Run:New Run" dialog box.

First thing; change the Run Name: to "First Tutorial". If you keep accepting the default "New Run" name, it won't take long to get lost in a pile of "New Runs".

Fuel Moisture Files

In the Fuel Moisture Files area of the Input tab click the Browse button . An "Open" dialog box appears, navigate to the C:\...FlamMap3\Tutorial\ folder and select the Broad.FMS file and click the Open button.

There are no custom fuel models in this landscape so you can ignore the Use Custom Fuels (*.fmd) option. Your Fuel Moisture Files area will look like this;

Winds

In the Winds group make sure the Wind Blowing Uphill radio button is selected and change the Wind Speed (MPH @ 20'): spin box to "35". Remember these are the winds 20 feet above the tops of the tree crowns.

With the Wind Blowing Uphill radio button selected the wind direction used in the fire behavior calculations will be based on the aspect theme for each cell in the landscape. Adjacent cells where sharp changes in topography occur may have very different wind directions, and thus very different fire behavior. Your Winds group will look like this;

Canopy Characteristics

Since Stand Height, Canopy Base Height, and Canopy Bulk Density are all included in the Tutorial.LCP landscape file, those spin boxes are dimmed in this area. If these themes are not included in your Landscape (.LCP) File they can be set globally (across the entire landscape) in this group box.

The Foliar Moisture Content (%): spin box default is 100, this is a realistic moisture content for conifer foliage during an above average fire season. The Canopy Characteristics area will look like this;

Fuel Moisture Settings

In the Fuel Moisture Settings group box make sure the Use Fixed Fuel Moistures from Fuel Moisture File radio button is selected. The remaining options in this group are related to using the fuel moisture model and will not be used here.

The Outputs Tab

You select which grids you want FlamMap to generate on the Outputs tab. In addition there are two processing options you need to be concerned with.

Processor Usage

Unless you have a computer with more than one processor chip, you can't change the Single Thread spin box setting. If you don't know how many processors your computer has, leave the Single Thread spin box set to the default, which is the maximum number your computer has.

Memory/Disk Usage

Leave the default Create outputs in memory radio button selected.

Output Grids

In the Output Grids group box select the following check boxes;

  • Rate of Spread (ROS)
  • Flame Length (FML)
  • Crown Fire Activity (CFR)

Since you are using fixed fuel moistures from the Fuel Moisture (.FMS) File, none of the Fuel Moisture Outputs are available.

Non-Fixed Fuel Moisture Outputs

These outputs are available only when a conditioning period is used, this is covered in the Variable Fuel Moistures advanced lesson.

Crown Fire Calculation Method

Leave the Finney (1988) radio button selected. More information on this option is found in the help system.

Options

Make sure the default Relative Spread Direction from Maximum radio button is selected in the Options group box.

Now your Output tab should look like this;

Almost Ready to Run

Since you have selected options that will generate a valid FlamMap run, the Apply button is now available. Before any of the options in the "Run:" dialog box take effect in a FlamMap run, they need to be applied. Click the Apply button and the Launch button becomes available, while the Apply button is now dimmed and the messages on the status bar change. If you make any changes to the Inputs or Outputs tabs the Apply button again becomes available.

Click to OK button in the "Run:" dialog box to close it, if you have previously clicked the Apply button all the settings in the "Run:" dialog box will be saved.

Save the Project

Even if you plan to continue now with this tutorial, you should save this project by following the procedures in the first lesson. Keep the project name Tutorial.fmp.

The next lesson will finally generate maps in a FlamMap run.
