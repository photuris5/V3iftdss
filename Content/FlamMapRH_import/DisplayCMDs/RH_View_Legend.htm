Legend

The legend is an important feature of FlamMap, not only does it provide a reference of displayed colors but is used to;

  • control what colors are displayed,
  • change units,
  • modify the ranges and classes of the displayed theme.

Viewing the Legend

You open the "Legend" dialog box with the View > Legend command or by choosing View Legend from the shortcut menu that appears when you right-click in the "Display" pane.

The legend displayed is that of the active grid theme. The initial colors, units, decimal places, and number of classes are set in the "Preferences" dialog box.

You can move the "Legend" dialog box anywhere on your screen (even outside the "FlamMap" window) by dragging it by it's title bar.

You can change the units displayed in the legend and "Display Pane" by selecting one of the choices from the Units: drop-down list.

 

 

By expanding the "Legend" dialog box or using the scroll bars you can view the entire grid theme legend and any visible auxiliary vector and contour theme legends. Specialized point vector themes, such as wind and spread vectors, are not shown on the legend.

 

Editing the Legend

You can quickly change the display units of the active grid theme by selecting one of the choices from the Units: drop-down menu. For more modifications to the active grid theme legend click the active theme button above the grid legend to display the "Create/Modify Legend" dialog box.

Clicking one of the vector theme buttons in the "Legend" dialog box will display the "Properties" dialog box where you modify the way vector themes are displayed.
