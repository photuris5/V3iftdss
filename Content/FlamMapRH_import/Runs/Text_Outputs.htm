Text Outputs

Three text outputs, Fire Size List, FLP - 20 Bin Metric, and FLP - 6 Bin English are available from a MTT Burn Probabilities run. After a run is completed expand the run tree item by clicking it's box to show the text outputs.

Right-clicking any individual output grid item in the "Tree" Pane displays this short cut menu.

Save As...

Selecting the Save As… command opens a standard Windows "Save As" dialog box where you can navigate and name your new Text (.TXT) File.

Open

Selecting the Open command from the shortcut menu open the text outputs with the computer's default text editor. These outputs can be very large and if the default text editor is Notepad it can take a while to open or it could even crash FlamMap. It is recommended to first save the outputs as a Text (.TXT) File using the Save As... command, then open the file with a third party text editor or WordPad which can handle larger files easier.

Help

Selecting the Help command will bring you to the Output Grids heading above in this online help system.
