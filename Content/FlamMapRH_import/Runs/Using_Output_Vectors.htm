Using Output Vectors

FlamMap can create several types of vector outputs depending on the type of run;

 

Run Type

Vector Outputs

Fire Behavior Outputs Tab

Spread Vectors, Max Spot Vectors

Minimum Travel Time Tab

Flow Paths, Major Paths, Arrival Time Contours

Treatment Optimization Tab

Flow Paths, Major Paths, Arrival Time Contours

All Runs

Wind Vectors

Vector outputs are selected from the Outputs sections of the Fire Behavior Outputs, Minimum Travel Time, and Treatment Optimization Model tabs in the "Run:" dialog box. The Wind Vectors are always created when using one of the Gridded Wind options.

Manipulating Vector Outputs

To temporally hide or display an individual vector output theme, simply double click it's line in the "Tree" pane. A blue check mark "" indicates the theme is currently displayed and a red "X" indicates it is currently hidden.

Right-clicking a individual vector theme item displays the following shortcut menu. With one exception (see below) this is the same menu and commands used to manipulate Auxiliary Vector and Contour Themes. With these commands you can delete an individual theme, save as a shapefile, or change the way an output theme displays.





Exceptions for Spread, Wind, and Max Spot Vectors

Spread, Wind, and Max Spot Vectors are point shapefiles and contain attributes (in the .dbf file) that are used in the display. Thus the display properties and legending of these shapefiles is different from other shapefiles in FlamMap. When you select the Properties command for one of these Output Vector Themes the corresponding "Vector Legend" dialog box opens from which you control the display of these unique themes.
