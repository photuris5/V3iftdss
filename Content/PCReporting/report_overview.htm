Data Reporting

Data can be sent to databases from within the user interface. Currently there are two databases linked to :

Monitoring Trends in Burn Severity (MTBS)

The MTBS is designed to consistently map the burn severity and perimeters of fires across all lands of the United States from 1984 and beyond.

  • Visit the MTBS help topic for more information about reporting MTBS data with
  • For more information about the MTBS system visit MTBS.gov
 Interagency Fuel Treatment Effectiveness Monitoring (FTEM)

The Interagency Fuel Treatment Effectiveness Monitoring database provides is source for information to verify fuel treatment effectiveness. It also establishes baseline data for fuels program decisions.

  • Visit the FTEM help topic for more information about reporting to FTEM within

The IFTDSS development team is working behind the scenes to build out the Reporting capabilities. These functions will become available in a future release.
