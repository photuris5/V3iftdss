MapStudio Tips and Tricks

 

Here are some answers to common questions about using

Creating an area of interest with the help of reference data?

The Reference Data can be used to create your own ‘Area of Interest’ shapes for reports, and ‘Landscape Masks’ for editing using the Create/Edit Tool.

 Folders, saving a step

When creating a new landscape in Map Studio using Landscape Tools, be sure to first create a folder for it My Workspace. This will keep you from having to pause your mapping to create a folder, and will help keep your workspace uncluttered as you add more project areas.

 Landscape file, where is it?

After you add data, landscapes, simulation outputs, or reference data, you need to use the Layer List to turn them on and off and see the legend.

 Landscape Size, how large can landscapes be?

Landscapes may be a maximum of 12,188,160 acres

 Legend, where is it?

To view the 'legend' for your layers, click Layer List . You may turn layers on by selecting them in Landscape Tools , or going to My Workspace, selecting your file, and using the View on Map button that drops down when a file is selected.

Reference layer, I turned it on but don't see it

If you don’t see a reference layer when you first turn it on, you may need to zoom in for it to appear.

 Shapefiles disappeared, how can I permanently save them?

The process for saving shapefils permanently is detailed in the Saving Shapefiles topic.

 Swipe Tool, which layer is shown at the top and bottom?

When using the Swipe tool, the layer you select in the dropdown list will be the one shown in the top half of the swipe screen.
