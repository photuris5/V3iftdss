Adding Reference Data

Add Layers

With Add Layers you may add shapes or shapefiles you have created or uploaded to to your Layer List, or add reference data such as agency boundaries, wildfire activity, and other information.

Add Shapes or Shapefiles
  1. Click Add Layers.
  2. Select the My Layers tab which displays all of your shapes and shapefiles organized by the folder in which they are stored.

  3. Select the folder for the file you desire.

  4. Click the checkbox next to the shape you need.

  5. The map will zoom to your shapefile and it will be available in Layer List.

 Add Reference Layers
  1. Click Add Layers
  2. Select the Reference Layers tab which displays all the reference layers available in . Scroll to the bottom of this page for a full list of current Reference Layers.
  3. Select the drop-down for the type of data you require.

  4. Use the checkbox next to the specific data you require.

  5. Your data will be displayed in and on Layer List.

You may add reference data layers to that outline agency boundaries, wildfire activity, and other information.

To add a reference layer to your Map:

  1. Click Add Reference Data
  2. Click the drop-down arrow next to a Data category to view available data.

  3. Click the check box indicating the data you wish to display.

  4. When finished, you may close the Add Reference Data box, your added layers will be added to Layer List.

For a legend of the different boundaries click the Layer List icon at the bottom of the map, then click the drop-down arrow to the right of the layer checkbox. This will drop down with a legend indicating each data layer on your map. Visit the Layer List Topic to learn more about that mapping feature.
