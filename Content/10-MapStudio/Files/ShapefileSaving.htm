Copy and Save a Shape from Another Source in

When you want to copy and save shape from elsewhere in , for example the administrative area for a forest from Reference Layers. To do this in :

  1. With the shape you wish to copy open, zoom to the approximate area of .

  2. Click the Create/Edit Shapes widget to open it.

  3. Click Copy from Existing Features

  4. Select the appropriate setting: "Select singlepart" if you are copying a single shape. Use "Select multipart" if you are copying a multipart shape. If you are copying several single part shapes to create a group, use "Select singlepart", click the first shape, click Copy from Existing Features again, ensure multipart is still select, and click the second shape. Repeat as many times as necessary until all your desired shapes are selected.

  5. Click the shape you wish to copy. When selected, the shape will be highlighted in yellow, and additional fields will appear inCopy from Existing Features for you to populate
  6. Give the shape a name.
  7. Select the folder in which to save the shape. You may use the Add folder button if you need to create a new folder here.

  8. Click Create.

  9. When successfully copied, your new shape will appear in dashed lines in , and a confirmation will appear at the top of Create/Edit Shapes. The shape will now be available in My Workspace, and throughout .

Saving Uploaded Shapefiles for Future Use

Follow the steps below to permanently upload and save a shapefile in My Workspace for later use in . Without this step, navigating away from results in an uploaded shapefile not being saved. This is a known issue that will be corrected in a later release.

Upload the Shapefile

Use Add Shapefile to upload a shapefile to :

  1. Click the Add Shapefile widget along the bottom of to open Add Shapefile.

  2. Click Upload File. A browser box will open allowing you to search your local files. Select the desired shapefile. Shapefiles must be zipped to be uploaded and contain .shp, .shx, .dbf., and .prj files.

  3. Use the checkbox to select the attributes you wish to import. You may choose up to 10.

  4. When finished, click the Add to Map button.

  5. When completed, a confirmation message will appear to verify the shapefile has been uploaded.

Save the Shapefile

To permanently save your uploaded shapefile:

  1. To select the new shapefile, exit Add Shapefile if its open, then open Layer List
  2. Zoom to your new shapefile.
    1. Click the More Options button to the right of your shapefile and

    2. Click the Zoom to option. This will zoom to your newly uploaded shapefile.

  3. Open Create/Edit Shapes

  4. When the Create/Edit Shapes box opens, you can choose to either save a single shape, or multiple shapes:

    Save a single uploaded shapefile:
    1. Click the Copy from Existing Features button ,
    2. Make sure the Select Single Part option is selected
    3. Select the polygon you wish to save (selected polygons will turn yellow). This can be done by either left clicking the individual polygon.
    4. Give your polygon a descriptive name.
    5. Choose a folder in which to save your polygon.

    6. Click Create.

         Save multiple polygons:
    1. Click the Copy from Existing Features button ,
    2. Make sure the Select multipart option is selected, then
    3. Left mouse click your first polygon, you will notice its color turns to yellow.
    4. Next click Copy from Existing Features button again and left click on polygon two, repeat these steps for each polygon until they are all outlined in yellow.
    5. Next, type a name for your polygons in the edit box.
    6. Specify the folder to save the set in.

    7. Click Create.
