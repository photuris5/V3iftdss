Creating a Landscape

The first step in working with landscapes in is to create a landscape and evaluate the information. In , a landscape file refers to the spatial file that contains vegetation and fuel data in .lcp file format.

You may create a landscape using LANDFIRE data, either by using and the Landscape Tools widget, or through the Landscape Summary task in the Landscape Evaluation Stage of the Planning Cycle. Instructions for each are outlined below:

In future releases of you will be able to upload your own .lcp files

Create a new landscape in Create a new landscape in the Landscape Summary task of the Planning Cycle using the "Create Here" button

Once you click Create, you will be returned to the Landscape Summary task, and the field next to My Landscapes, displays the name of your newly created landscape, with an hourglass next to it, indicating it is being created. Once created, the hourglass is replaced by a green checkmark. You may have to use the refresh button next to the landscape name to see the green checkmark.

 To create a new landscape in the Landscape Summary task of the Planning Cycle using the "Create in Map Studio" button

If you prefer to create a landscape in , use the Create in button.

This option will automatically take you to , where you will follow the same steps outlined in the instructions above. When you are finished in , you will have to navigate back to the Landscape Summary interface.
