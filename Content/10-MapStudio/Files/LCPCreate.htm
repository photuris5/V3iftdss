Creating a Landscape

The first step in working with landscapes in is to create a landscape and evaluate the information. In , a landscape file refers to the spatial file that contains vegetation and fuel data in .lcp file format.

You may create a landscape from LANDFIRE data, either by using and the Landscape Tools button, or through the Landscape Summary part of the Landscape Evaluation Stage of the Planning Cycle. Instructions for each are outlined below:

In future releases of you will be able to upload your own .lcp files

To create a new landscape in

Once you click Create a message will appear at the top of the Landscape Tools box to confirm your landscape is being created

It may take a couple minutes before your landscape is available for viewing. When the landscape file is fully created, you will be notified via the Message Center in the top right of your screen.

 To create a new landscape in the Landscape Summary task of the Planning Cycle using the 'Create Here' button To create a new landscape in the Landscape Summary task of the Planning Cycle using the 'Create in Map Studio' button

To leave this interface and generate a landscape in , click the Create in Map Studio button.
