Add a Shapefile

  • Use the Add Shapefile to add a shapefile in .
  • Select Upload File.
  • Choose your zipped shapefile

  • To save the shapefile, make sure to check Save result to my workspace.
  • When finished, click Add to map.
  • To add a shapefile to it must contain shp, .shx, .dbf, and .prj files, and be in a compressed (zipped) file format. Visit the About Shapefiles topic for full details on shapefiles in .
