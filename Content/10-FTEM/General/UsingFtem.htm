How to Use FTEM

The FTEM interface is arranged to walk you through the treatment monitoring process. Use the tabs to select a wildfire, select your treatments, enter monitoring data, and review and complete your monitoring. Below, we walk through each stage of this process. The following steps assume you have an FTEM account. If not, visit the FTEM Setup Topic for instructions. Upon opening FTEM, you will see wildfires with a split view of on the left, and a table view on the right.

By default, the interface is split, displaying a map on the left, and monitoring tabs on the right. Change the interface at anytime to show only the tabs or only the map by clicking the split screen buttons in the top of the screen just below the top navigation menu.

 Step 1: Pick Your Wildfire

Pick a wildfire based on year and wildfire name:

  1. Enter a Wildfire Year, the year the wildfire was declared under control, or extinguished.
  2. (Optional) If you are affiliated with multiple agencies, filter your results by agency. If you are not an administrator or dual agency editor, this option may not be visible.
  3. (Optional) Use the wildfire search field to further constrain your results by entering the fire name.
  4. Click to select the desired fire name from the list.

    5. Need to de-select your fire? Move the mouse cursor to the Wildfire tab and click the black X to the left of the wildfire name.

  5. (Optional) Click the Zoom To button to zoom to the fire on the map, or the Table Summary button to see a summary of the interactions for that fire.

  6. Proceed to the Treatments Tab. You may also proceed to Treatments by clicking the Select Treatments button located to the left of the Zoom To button.

Attached files can be added or downloaded in later tabs (see the Attachments topic). Notice that each wildfire listed is followed by information, left to right, describing the number of acres, control date, monitor status, and the number of interactions.

 Step 2: Select Treatments

In this step you will choose all the treatments for your agency that you know interacted or influenced the fire to form your list of interacting treatments. You will add monitoring data for the listed treatments via the Monitoring tab. You can think of this treatment select step as your "shopping cart" where you place the treatments that require monitoring, and remove the ones that don't.

  1. If you know there are treatments that interacted with the wildfire, but do not see them here, use the Add Treatment(s) button to search and select treatments. This button allows you to search a list of all treatments within a specific buffer distance from the fire.
  2. If there are treatments that did not interact with the fire, use the check box to select them and use the Remove Treatment(s) button. Removed treatments may be added back to the list using the Add Treatments(s) button.
  3. Once you have all the necessary treatments, select them using the chekboxes, or using the Select All button.

    Note the colored icons to the left of the treatment names. In this example the polygon icons indicate these treatments are polygons, and their red color indicates monitoring/data entry for them has not been started. Learn more about these symbols by visiting the FTEM Symbology Topic.

  4. Proceed to the Monitor tab either by clicking Monitor Treatment(s) button at the bottom right of the list.

Notice that each treatment listed is followed by information, left to right, describing the treatment ID, type of treatment, date of completion, agency, number of acres, control date, jurisdiction unit, and the number of interactions.

 Step 3: Enter Monitoring Data
  1. Select a treatment to monitor using the checkbox next to it

  2. Click the Enter Data button. to start monitoring the treatment

    By using the check box for one or more treatment, you may choose to enter data for treatments individually, or in batches. Note that when running batches, some fields will still need to be revisited and filled individually for each treatment such as time of burn and acres burned.

  3. A series of questions for the treatment(s) interaction appears as a series of drop-down headings addressing specific monitoring questions. The first header, "Interaction Details and Treatment Effects" is required. Enter data into each field. Guidance for monitoring each required field is described in the FTEM Required Questions Guidance Topic.
  4. Proceed to any optional fields and enter data. Guidance for monitoring each optional field is described in the FTEM Optional Questions Guidance Topic.
  5. Once your data is entered, click Save and Close to proceed. If you want to save your partially completed work and resume the rest of the process later, click Save.

    6. Note that in this example two of three treatments are selected for data entry. You can choose to enter the same data for all your treatments, or like this example, enter data for some treatments, and later enter different data for other treatments. This can be useful if multiple treatments were affected, but burned under different conditions.

  7. Proceed to the Complete tab.

 Step 4: Complete the Process by Reviewing and Submitting your Data

In the Complete tab:

  1. Click the Table Summary button to generate a review table of treatments and data entered. You may also review attachments using the Attachments button. At this point, if you are satisfied with your treatment data, you may proceed to Step B. However, if you find a data entry that needs editing or additional information, you can use the tabs to navigate back, select treatments, and modify data.

  2. Click the Complete Wildfire Monitoring button to submit your data to the FTEM System.

At the bottom of the tab, the number of "completed", "in progress", and "not started" treatments displays.

Note the Complete Wildfire Monitoring is grayed out until all treatments are completed.
