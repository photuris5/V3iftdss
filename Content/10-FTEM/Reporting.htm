FTEM Reporting allows you to generate summary reports of fuel treatment effectiveness monitoring data for specific fires, dates, and agencies. Report types include:

  • Wildfire Specific FTEM Report
  • Agency Specific FTEM Report
  • Long Report - Agency, Region, Wildfire
  • Long Report - State, Wildfire
  • Data Dump-csv/xlsx of all data
  • FTEM Adhoc Report

To generate a summary report:

  1. From the FTEM mainpage, click the FTEM Reporting button.

  2. Select your report type.
  3. Select the year the wildfire occurred.
  4. Select the agency jurisdiction for the wildfire.
  5. (Optional) Search the list of available wildfires for a specific name.
  6. Select the wildfire from the drop-down list.

  7. Click Generate Report.

  8. A summary report will be generated and displayed in a new browser tab.
