Editing a Landscape to Develop Treatment Alternatives

In Developing Treatment Alternatives, your Originating Landscape must be an Extracted, unedited LANDFIRE landscape and the only landscapes available for you to edit must fall within the same landscape extent.

Getting Ready to Edit

First, select your Originating Landscape to edit from the dropdown menu.

 Add Default Fuels Treatment/Disturbance Rule

Add Landscape Edit Rule

Applying Landscape Edits

Once your rules are selected, give the new edited landscape a descriptive name and select Create New Landscape.

Edited landscapes will be created as new files, this ensures you will always have the original landscape for comparison. You may repeat this process to create your Existing Condition landscape if you need one, and the treatment alternative landscapes.

Edited landscapes are available in My Workspace, stored by default in the same folder as the landscape you are making edits to.

When clicking on landscape files that have been edited in My Workspace, you will see an option to View Edit Rules. This displays the edits that were made on that landscape.
