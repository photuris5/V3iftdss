Publishing

The TOC is the 'IFTDSS Online Output (Master)

The publishing Target is in the Online Output folder, 'HTML5 - Top Navigation'

Pushing the project to GitHub for IBM

In the target, go to the Publishing section of the target, select New Destination, then specify the desired folder on your local machine that you would like to store a copy of the output file.

Push that local folder to:

https://github.com/hyde1520/iftoutput

Publishing the project to keep our working version current

To update the project:

Begin by saving your project like usual

Next, go to Source Control and select Commit All.
