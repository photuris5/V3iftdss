Navigating

Once you have successfully logged into , you will be will see the Wheel.

The wheel will guide you through the various tasks is designed for. Tasks include:

  • Landscape evaluation
  • Strategic Planning
  • Implementation Planning
  • Monitoring
  • Reporting

To skip these walk-throughs and proceed directly to the map, click 'Take me to the Map'

You may return to the wheel page at any point in by clicking the wheel icon in the top left portion of the menu bar.

 

The menu bar, at the top of the page, remains visible no matter where in you are.

The menu bar includes from left to right:

  • Home: Returns you to the main wheel page
  • My Workspace: The place in where all the landscape files are stored
  • Map Studio: The mapping interface of , view data, edit data, add reference layers, and select landscape tools
  • Playground: The place in where all the model run outputs are stored
  • Help: Access to the help system, about page, and feedback button
  • User Profile: Displays where user profile you are logged in as

Map Studio

The map studio is where the bulk of your landscape editing and review will take place.

In the top left:

  • Zoom In and Out: Zoom the map extent
  • Default extent: Return the map to its default extent

In the top right: 

  • Identify: Identify a feature
  • Draw: Draw lines and shapes on the map. This is for display only, it does not create a shapefile
  • Swipe: Allows for quick toggling between different layers

In the bottom right:

  • Attribute Table: Displays a table of map features

In the bottom center:

  • Legend: A legend of the map layers, displays on the left of the screen.
  • Layer list: A list of the layers shown on the map
  • Add reference data: Addition of federal reference data layers such as administrative boundaries.
  • Add shapefile: Allows a user to upload a shapefile to the map.
  • Landscape tools: Allows you to add an existing landscape, create a new landscape, or modify an existing landscape in the map studio
  • Simulation output tools: Allows you to add simulation outputs to the map, or compare results from multiple outputs.
  • Basemap gallery: A selection of basemap layers that can be added to the map.
  • Measurement: Allows you to measure area or distance in map studio.
