Creating a landscape

Landscapes in can be created in multiple ways. For example, you can create a landscape in using the Landscape Tools button . You can also create a landscape from most of the tasks, using the Create Here button as displayed in the example below.

For this example, we’ll create a new landscape using that Create Here button from the Landscape Summary Task in . This serves two purposes, to create a landscape, and to do so in the interface that allows you to easily create a landscape summary if you so choose (covered in the next tutorial).

To get to the Landscape Summary interface:

  1. Navigate to the Planning Cycle using the top navigation menu.
  2. Select the Landscape Evaluation stage of the Planning Cycle.
  3. Click the Landscape Summary task.

From the Landscape Summary page, click Create Here:

Next, you'll go through the steps of creating a landscape. As you go through the process below, use the following values:

  • Location: Tahoe National ForestWest: -120.8856 degrees, East: -120.6044 degrees, South: 39.4463 degrees, North: 39.6149 degrees
  • Landfire version: 2012
  • Fuel Model: Fuel Model 40
  • Landscape Name: HaypressCrk LF2012 Unedited
  • Folder: Specify or create: Haypress Creek Project

To help keep track of your data, we recommend including the LANDFIRE version in the name.

Once you click Create, you will be returned to the Landscape Summary task, and the field next to My Landscapes, displays the name of your newly created landscape, with an hourglass next to it, indicating it is being created. Once created, the hourglass is replaced by a green checkmark. You may have to use the refresh button next to the landscape name to see the green checkmark

For the rest of your acitivies in this landscape will be available in the folder you specified in My Workspace, and access it in drop-down menues throughout .
