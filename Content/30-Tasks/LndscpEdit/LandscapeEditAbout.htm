Editing Landscape Data for Landscape Evaluation

Getting Ready to Edit

First, choose a landscape to edit from the Starting Landscape dropdown menu.

 Add Default Fuels Treatment/Disturbance Rule

The first step in applying a Default Fuel Treatment / Disturbance Edit Rule is to specify the Treatment or Disturbance Type, and when prompted, choose the intensity or severity of the fuel treatment or disturbance type.

Next you will choose the Time Since Treatment or Disturbance that is most appropriate.

Lastly you will decide on the extent to which you will apply the rule. If you choose a Landscape Mask, it allows you limit the application of the rule to only the area within the chosen mask. If Landscape Mask is left blank rule will be applied to the entire landscape.

 Add User Created Edit Rule Applying Landscape Edits

Once your rules are selected, give the new edited landscape a descriptive name and select Create New Landscape.

Edited landscapes will be created as new files, this ensures you will always have the original landscape for comparison.

Edited landscapes are available in My Workspace, stored by default in the same folder as the landscape you are making edits to.

When clicking on landscape files that have been edited in My Workspace, you will see an option to View Edit Rules. This displays the edits that were made on that landscape.

You may find the Swipe tool helpful in comparing landscape edits.
