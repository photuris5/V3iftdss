1) Adjust the treatment list so it reflects the treatments that need monitoring.

2) If treatments are visible that do not need monitoring, select them using the checkbox and click the Remove Treatments button.

3) If treatments are not visible that need monitoring, add them using the Add Treatments button.

4) Once the list is populated only by treatments that need monitoring, and all are checked, progress to the Monitor tab.
