You also may navigate using the menu bar

About the menu bar

The menu bar, at the top of the page, remains visible no matter where in you are.

The menu bar includes from left to right:

  • Home: Returns you to the Planning Cycle
  • My Workspace: The place in where all the landscape files are stored
  • : The mapping interface of , view data, edit data, add reference layers, and select landscape tools
  • Playground: The place in where all the model run outputs are stored
  • Help: Access to the help system, about page, and feedback button
  • Messages and Warnings: displays status updates and warnings on landscape or modeling operations
  • User Profile: Displays user name and links to access or edit your user profile

To skip these walk-throughs and proceed directly to the map, click 'Take me to the Map'
