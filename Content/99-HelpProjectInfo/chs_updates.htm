Generating new identification numbers for context sensitive help

Generating new topic identification numbers for context sensitive help is done in the Alias file, under the Project folder and Advanced folder.

Click the Generate Identifiers for this Project button

  1. Ensure the Identifiers.h file is selected under 'Choose Existing'
  2. Make sure 'Include topic name in identifier name is checked, and 'Capitalize identify names' is unchecked
  3. Make sure Unassigned topics only is selected under 'Generate Identifiers for'
  4. Select Create.

This will generate new identifier numbers in the Identifiers.h for any new topics so they can be linked within the application if needed.
