Account Management for FTEM Administrators

An FTEM Administrator role allows you to manage accounts for your agency by approving new users, or editing the roles for existing users. FTEM Administrators may also delete monitoring interactions if needed. Click on the topic below for instructions on how to perform these functions:

Approving a New FTEM User

FTEM Administrators will be notified via email when a user requests access to FTEM, or when they need to change their user role.

To approve a User's FTEM request, you must first be logged into your FTEM account and access account information from any FTEM page:

  1. From any page in FTEM, click your name in the top right.

  2. Select the User List from the drop-down.

  3. Click on the row of the User List that contains the requester's User name.

  4. Click the Grant FTEM Access button. This will open an approval dialogue box to complete the approval. You also have the option to deny the request from here.

  5. To finalize the account approval, briefly review the information in the dialogue box, and if editor role was requested approve the request by checking the box next to "Grant editor role". When ready, finish by clicking the Grant Access button.

 Search User List

Click on the categories on the left of the User List to filter the list of users based on their display/status (account pending or account disabled), or agency. Note that you may combine filters, (e.g. agency and display in most cases. However, when using the 'Disabled User' option, it will not further filter by agency.

 Changing a Users Role

FTEM Administrators will be notified via email when a user requests a different role in FTEM.

To Change a User's FTEM role, you must first be logged into your FTEM account and access account information from any FTEM page:

  1. From any page in FTEM, click your name in the top right.

  2. Select the User List from the drop-down.

  3. Click on the row of the User List that contains the requester's User name.

  4. Click the Grant Agency Roles button. This will bring up a table in which you can select different roles for various agencies.

  5. Use the checkboxes to grant the appropriate roles for a User.

    6. Unclear about the different User Roles? See the User Roles Topic.

  6. When complete, click the Grant Roles button.

  7. The user will be automatically sent an email to inform them of the role change.

 Check the Monitoring History of a Fire

To see the entire system history for monitoring within a wildfire:

  1. Go to the Wildfire tab and choose your wildfire.

  2. Click the Audit Log button.

  3. An exportable report will open detailing all information changes, and which user entered them.

 

 Amending Monitoring Data

As an FTEM Administrator, you may need to amend or edit monitoring data.

To amend a treatment

  1. Locate the fire on the Wildfire tab and click on it.
  2. Skip to the Complete tab.

  3. Click the Amend Wildfire button.

  4. Provide the reason for your amendment when prompted.

  5. Navigate to the Monitoring or Treatment tabs and update with data as needed.
  6. When finished, return to the Complete tab and click the Complete Wildfire Monitoring button.

  7. The wildfire treatment monitoring will now appear with a version number to indicate the update.
