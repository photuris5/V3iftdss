Navigating IFTDSS

Once you have successfully logged into IFTDSS, you will be will see the Wheel.

The wheel will guide you through the various tasks IFTDSS is designed for. Tasks include:

  • Landscape evaluation
  • Strategic Planning
  • Implementation Planning
  • Monitoring
  • Reporting

To skip these walk-throughs and proceed directly to the map, click 'Take me to the Map'

Note: you may return to the wheel page at any point in IFTDSS by clicking the wheel icon in the top left portion of the menu bar.

The menu bar, at the top of the page, remains visible no matter where in IFTDSS you are.

The menu bar includes from left to right:

  • IFTDSS Home: Returns you to the main wheel page
  • My Workspace: The place in IFTDSS where all the landscape files are stored
  • Map Studio: The mapping interface of IFTDSS, view data, edit data, add reference layers, and select landscape tools
  • Playground: The place in IFTDSS where all the model run outputs are stored
  • Help: Access to the IFTDSShelp system, about page, and feedback button
  • User Profile: Displays where user profile you are logged in as
