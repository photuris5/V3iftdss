Landscape (.lcp) Files

A Landscape file (.lcp) is a multi-band raster format commonly used by wildland fire behavior and fire effects models such as FARSITE and FlamMap. The bands of an .LCP file store data describing terrain, tree canopy, and fuel model. The .lcp files used by IFTDSS contain information describing:

  • Aspect
  • Canopy base height
  • Canopy bulk density
  • Canopy cover
  • Canopy height
  • Elevation
  • Fire Behavior Fuel Model (FBFM 13 or 40)
  • Slope

In IFTDSS, a Landscape file is the basic file needed to define your project area and run most of the modeling operations. In the IFTDSS application, and in many places throughout these help documents, we refer to "creating a .lcp file". During this process, you are actually downloading the .lcp file for your chosen area from LANDFIRE.

There are multiple places in IFTDSS where Landscape files from which Landscape files may be created. Visit the Create Landscapes Topic for a list of methods and step by step details.
