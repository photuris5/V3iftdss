Creating a Landscape

The first step in working with landscapes in IFTDSS is to create a landscape and evaluate the information. In IFTDSS, a landscape file refers to the spatial file that contains vegetation and fuel data in .lcp file format.

You may create a landscape using LANDFIRE data, either by using Map Studio and the Landscape Tools widget, or through the Landscape Summary task in the Landscape Evaluation Stage of the Planning Cycle. Instructions for each are outlined below:

In future releases of IFTDSS you will be able to upload your own .lcp files

Create a new landscape in Map Studio

To create landscape from within Map Studio click the Landscape Tools widget along the bottom of Map Studio. In the Landscape Tools box, make sure the Create new tab is selected.

In the Create new tab specify the requirements for your landscape:

  1. Use the Map Studio tools such as zoom, basemap layers, etc. to locate your desired area.
  2. Click the Select draw mode, mouse to your location on the map, hold down the left mouse key, and draw a rectangle around your desired area, releasing the button when you are finished.
  3. Alternatively, if you wish to enter exact coordinates rather than draw a box around your area, you may do so by entering the coordinates, in degrees, for the west, east, south, and north bounds of your desired area.

    4. The maximum area for a landscape is 12,188,160 acres (138 miles x 138 miles).

  4. Choose the version of LANDFIRE data you would like to use.
  5. Choose the fire behavior fuel model (Fuel Model 40 or Fuel Model 13) you would like included in your landscape file.
  6. Give your landscape a descriptive name, this will be helpful as you accumulate more landscape files in IFTDSS.
  7. Select the folder in which you would like to store your landscape file.

  8. Click Create to generate your landscape file.

Once you click Create a message will appear at the top of the Landscape Tools box to confirm your landscape is being created. it may take a few minutes to generate your landscape. You may check its status by refreshing the Layer List in Map Studio.

If you have created a landscape in Map Studio using the Create in Map Studio button (for example, this is an option in Landscape Summary, and other tasks) you will need to navigate back to your task to continue that process using your newly created landscape.

Create a new landscape in the Landscape Summary task of the Planning Cycle using the "Create Here" button
  1. Use the bounding box fields and enter the coordinates, in degrees, for the west, east, south, and north bounds of your area.

    2. The maximum area for a landscape is 12,188,160 acres (138 miles x 138 miles).

  2. Choose the version of LANDFIRE data you would like to use from the drop-down list
  3. Choose the fire behavior fuel model (Fuel Model 40 or Fuel Model 13) to include in your landscape file from the drop-down list.
  4. Give your landscape a descriptive name, this will be helpful as you accumulate more landscape files in IFTDSS.
  5. Select the folder where you would like to store your landscape file, or click + Folder to create and name a new folder to place your landscape file.

  6. Click Create to generate your landscape file.

Once you click Create, you will be returned to the Landscape Summary task, and the field next to My Landscapes, displays the name of your newly created landscape, with an hourglass next to it, indicating it is being created. Once created, the hourglass is replaced by a green checkmark. You may have to use the refresh button next to the landscape name to see the green checkmark.

To create a new landscape in the Landscape Summary task of the Planning Cycle using the "Create in Map Studio" button

If you prefer to create a landscape in Map Studio, use the Create in Map Studio button.

This option will automatically take you to Map Studio, where you will follow the same steps outlined in the Map Studio instructions above. When you are finished in Map Studio, you will have to navigate back to the Landscape Summary interface.
