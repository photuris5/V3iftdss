The Add a Shapefile

Use the Add Shapefile to upload a shapefile to the IFTDSS Map Studio.

Follow the steps below to permanently upload and save a shapefile in My Workspace for later use in IFTDSS. If you do not also save the shapefile as outlined below, it will be lost upon navigating away from Map Studio. This is a known issue that will be corrected in a later IFTDSS release.

Upload the Shapefile

Use Add Shapefile to upload a shapefile to Map Studio:

  1. Click the Add Shapefile button along the bottom of Map Studio to open Add Shapefile.

  2. Click Upload File. A browser box will open allowing you to search your local files. Select the desired shapefile. Shapefiles must be zipped to be uploaded and contain .shp, .shx, .dbf., and .prj files.

  3. Use the checkbox to select the attributes you wish to import. You may choose up to 10.

  4. When finished, click the Add to Map button.

  5. When completed, a confirmation message will appear to verify the shapefile has been uploaded.

Save the Shapefile

To permanently save your uploaded shapefile:

  1. To select the new shapefile, exit Add Shapefile if its open, then open Layer List
  2. Zoom to your new shapefile.
    1. Click the More Options button to the right of your shapefile and

    2. Click the Zoom to option. This will zoom to your newly uploaded shapefile.

  3. Open Create/Edit Shapes

  4. When the Create/Edit Shapes box opens, you can choose to either save a single shape, or multiple shapes:

    Save a single uploaded shapefile:
    1. Click the Copy from Existing Features button ,
    2. Make sure the Select Single Part option is selected
    3. Select the polygon you wish to save (selected polygons will turn yellow). This can be done by either left clicking the individual polygon.
    4. Give your polygon a descriptive name.
    5. Choose a folder in which to save your polygon.

    6. Click Create.

    Save multiple polygons:
    1. Click the Copy from Existing Features button ,
    2. Make sure the Select multipart option is selected, then
    3. Left mouse click your first polygon, you will notice its color turns to yellow.
    4. Next click Copy from Existing Features button again and left click on polygon two, repeat these steps for each polygon until they are all outlined in yellow.
    5. Next, type a name for your polygons in the edit box.
    6. Specify the folder to save the set in.

    7. Click Create.
