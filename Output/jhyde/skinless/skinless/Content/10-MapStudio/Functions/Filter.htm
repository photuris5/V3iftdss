Filter

The Filter widget in Map Studio enables you to show specific shapefile and hide others based on criteria you specify. This can be useful if you are working in an area with many shapefiles (such as many treatments clustered in a small area, or overlapping each other). To choose which shapefiles are visible using Filter:

  1. Click on the Filter widget in the top right of Map Studio.

  2. Select the Edit Filter tab

  3. In the box, filter your results. This can be done by:
    • Specifying a folder
    • Typing a name to constrain your results
    • Choose from the list, checking the shapefiles you want to see, and unchecking the ones you want to hide:
    • Use the Select button to change which shapefiles are checked (check marks indicate which files will be visible)
    • To save time you will notice there is a Select All button and a
    • Deselect All button.
  4. Click Apply Filter.

This filter can be removed anytime by going back to the Filter button and selecting Remove Filter.

Filter is intended to limit your view to one or more shapefiles. To turn off all shapefiles, open Layer List and uncheck the shapes, shown below.
