The Planning Cycle - Monitoring

The Monitoring stage of the Planning Cycle will allow you to:

  • Enter Fuels Monitoring Data
    • Input Data to FTEM
    • Evaluate Using FTEM Standard Form
    • Access NFMD or MTBS Online Databases
    • Submit data to Monitoring Reporting Database FTEM (where warranted)
  • Enter Monitoring Data into FEAT
    • Input Data to FFI
    • Evaluate Data Using FFI Queries/Stats
    • Access NFMD or MTBS Online Databases
  • Revisit Goals and Objectives for Further Evaluation
