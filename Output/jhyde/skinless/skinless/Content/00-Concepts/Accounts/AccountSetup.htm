Setting Up an IFTDSS Account

To create a new account, start from the IFTDSS Login prompt: 

  1. Go to https://iftdss.firenet.gov/landing_page/

  2. Choose the Login option in the top right of the IFTDSS page and click Register at the bottom of the Login box.

  3. When the New User Registration page displays, enter the required information. Your User Name is displayed on many exportable IFTDSS reports, so pick one you don't mind sharing! When finished, click Register.

    When entering information into the "Justification" field, simply enter a sentence or less describing what you'd like to use IFTDSS for.

  4. After clicking Register you will see a confirmation message verifying your registration request has been submitted.

  5. Once your information is submitted, you receive an email from an IFTDSS administrator that contains an access code that you will use for your first time logging in. This code remains valid for 72 hours. You may now log into IFTDSS and set your password.

  6. When logging in for the first time, you may be prompted to accept the IFTDSS IT Security Rules of Behavior.

  7. After accepting the User Agreement you will prompted to enter your temporary access code one last time, and set up your password. When finished, click Set Password.
