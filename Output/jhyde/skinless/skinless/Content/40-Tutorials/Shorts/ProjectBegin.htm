Beginning a Project

If you have not done so already, create an IFTDSS account.

Creating an IFTDSS Account
Go to https://iftdss.firenet.gov/landing_page/Choose the Login option in the top right of the IFTDSS page and click Register at the bottom of the Login box.When the New User Registration page displays, enter the required information. Your User Name is displayed on many exportable IFTDSS reports, so pick one you don't mind sharing! When finished, click Register.When entering information into the "Justification" field, simply enter a sentence or less describing what you'd like to use IFTDSS for.After clicking Register you will see a confirmation message verifying your registration request has been submitted. Once your information is submitted, you receive an email from an IFTDSS administrator that contains an access code that you will use for your first time logging in. This code remains valid for 72 hours. You may now log into IFTDSS and set your password.When logging in for the first time, you may be prompted to accept the IFTDSS IT Security Rules of Behavior.After accepting the User Agreement you will prompted to enter your temporary access code one last time, and set up your password. When finished, click Set Password.

There is no formal definition of "project" for IFTDSS, rather, the term refers to the collection of landscape files, model runs, and associated data for your specific area. It is easier to keep track of your files in IFTDSS if you create a folder for your project first.

To create a folder:

  1. Go to My Workspace from the top navigation menu

  2. Create a folder by clicking the Create Folder option at the top of the My Workspace page.

  3. Give your folder a name, in this example "Haypress Creek Project", and click Submit:

  4. Your new folder will now be visible in the Folders list.
